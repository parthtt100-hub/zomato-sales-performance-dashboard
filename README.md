# Zomato Sales Performance Dashboard

## Project Overview
This Power BI dashboard analyzes Zomato sales data to uncover trends, customer behavior, and business insights.

## Key Metrics
- Total Sales
- Average Rating
- Total Restaurants
- Total Votes

## Key Insights
- Dining category contributes the highest revenue
- Sales peaked in April and dropped in May
- High-rated restaurants generate more customer votes

## Tools Used
- Power BI
- DAX
- Data Visualization
